# HapticFeedbackDemo
This is a demo of an environment which has haptic feedback created in Meta Haptics Studio

## What haptics scenarios did I choose?
1. Touching Grass
![image](HapticFeedbackStudio/Doggo.png)
It is triggered by colliding the collider of the right controller with the collider of the Grass. Only works with grass that has colliders.

2. Touching a Dog
![image](HapticFeedbackStudio/Doggo.png)
It is triggered by colliding the collider of the right controller with the collider of the Dog. 

3. Touching a Button
![image](HapticFeedbackStudio/Press.png)
It is triggered when the right controller is moved into the collider of the button and then released.
Also works by normal interaction such as trigger.

4. Picking up a Log
![image](HapticFeedbackStudio/Tree.png)
Is triggered by the right controller by using the grab button to grab the log.

## Feedback
1. Touching Grass

It felt immersive, but without an animation felt rather bland.

2. Touching a Dog

It was okay, but I felt like it could have been more random by using a sine wave or just syncing with the audio.

3. Touching a Button

It felt good, I liked having a feedback.

4. Picking up a Log

It felt like having a feedback that something was picked up, not that a log was picked up.
